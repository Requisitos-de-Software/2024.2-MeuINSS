# Lista de inspeção

## Introdução

Para garantir a qualidade na análise de qualquer artefato, é fundamental utilizar uma lista de verificação como base para os itens avaliados. Assim, a equipe elaborou uma lista de inspeção considerando os pontos destacado na etapa de modelagem de requisitos (parte 2) pelo professor da disciplina e os requisitos específicos do artefato a ser analisado nesta fase do projeto, visando maior precisão na avaliação do próprio grupo e do grupo+1 (grupo 07).

A lista foi organizada de forma que fosse avaliados itens de padronização da disciplina e também o conteúdo desenvolvido pelo grupo.
